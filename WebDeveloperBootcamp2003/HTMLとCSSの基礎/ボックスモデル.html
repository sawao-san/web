Borderテスト

Borderを設定しようとすると、最低でも以下のプロパティは定義する必要がある

  1. border-width : 枠線の幅（太さ）
  2. border-color : 枠線の色
  3. border-style : 枠線の見た目。実践ならsolid,点線ならdotted

仮に、枠線が設定された要素に先にWidthが設定されていて、枠線の太さを設定したときに 枠線の太さ分、幅が増えてしまうので、そういう場合は、以下のように記載する 
                    
                        p {
                            box-sizing: border-box;
                        }

paddingテスト

paddingは要素の内側の領域

marginテスト

marginは要素の外側の領域。
以下のボタンのスタイルには、paddingテストのスタイルで指定していた"padding"という文字をすべて"margin"と書き換えただけだが、こんなに違う

displayテスト

ブロック要素に対して、以下のように設定すると、インラインのように見せてくれる 

                    
                        display: inline

逆にインライン要素に対して、以下のように設定すると、ブロック要素のように見せてくれる

ただ、インライン要素はwidthとheightを無視し、上下に対するmarginやpaddingのスタイルが反映されないので、ブロック要素のdiv要素を横に並べると変になる。 

                    
                        display: inline

そこで、以下のように書くことで、ブロック要素の性質をもったinline要素を作り出すことができる 

                    
                        display: inline
