positionについて

以下の色が塗られた領域を移動させてみましょう

Static

Relative

これは、TopやLeftなどの方向を指定すると、その方向に空間が作成される

Absolute

親要素かもしくはその親、もしくは祖先の要素にPositionが設定されて入れば、その要素からの相対位置を指定する
祖先の要素全てにPositionが設定されていなければ、bodyから見た相対位置が設定される
（一見変わりないように見えるが、top 50pxを指定するとページの最上部から50px下に配置される

Fixed

画面表示されている領域を基準にして、どこに配置するかを設定することができる
このページの画面右上にずっと表示されているのはそれが原因

Sticky

画面表示した際には、relativeと同じだが、画面をスクロールすると、画面の表示領域からどれぐらいの位置にあるかを設定することができるオプション
style.cssのh1に設定したので、タイトルがずっとへばりついて来ていると思う
